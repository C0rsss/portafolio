## ✨ Sobre el proyecto

Portafolio personal diseñado y desarrollado **desde cero** con HTML, CSS y JavaScript puro. Sin frameworks, sin dependencias — solo código limpio y animaciones cinéticas.

El objetivo: crear una experiencia visual premium que refleje mi identidad como desarrollador, con tema oscuro/cyberpunk, efectos reactivos y un diseño cohesivo de principio a fin.

---

## 🎯 Características

| Feature | Descripción |
|---|---|
| 🖱️ Custom Cursor | Cursor personalizado con anillo magnético |
| ✨ Canvas Particles | Red de partículas animadas en tiempo real |
| 🔡 Scramble Text | Animación de texto al estilo hacker en el Hero |
| 🌀 Glitch Effect | Efecto glitch CSS en el título principal |
| 📜 Reveal on Scroll | Elementos que aparecen con IntersectionObserver |
| ⌨️ Typewriter | Frases rotativas con efecto máquina de escribir |
| 📱 Responsive | Mobile-first con hamburger menu en pantallas pequeñas |
| 📬 Formulario | Formulario funcional con [FormSubmit](https://formsubmit.co) |
| 🔢 Contadores | Números animados con easing suave |
